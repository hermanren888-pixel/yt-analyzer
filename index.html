⚠️ 登录弹窗闪退怎么办？
这是 Google 同意屏幕还在「测试模式」导致的。请按以下步骤操作：
  1. 打开 Google Cloud Console → OAuth 同意屏幕
  2. 在「测试用户」下方点击「ADD USERS」
  3. 添加你的邮箱：hermanren888@gmail.com
  4. 保存后再点击「立即登录」

或者在同意屏幕顶部点击「发布应用」→ 切换到生产模式（这样任何 Google 账号都能用）

  • 点击率 (CTR) — 展示次数中产生观看的比例
  • 平均观看时长 — 观众平均每次观看的时间
  • 展示次数 — 视频缩略图在 YouTube 上展示的次数
  • 观看时长(分钟) — 总观看时长
  • 订阅者变化 — 新增/流失订阅数

配置步骤：

  1. 前往 Google Cloud Console
  2. 启用「YouTube Analytics API」
  3. 创建 OAuth 2.0 客户端 ID（Web 应用类型）
  4. 在「已获授权的 JavaScript 来源」中添加 http://localhost:8765
  5. 将生成的 Client ID 填入上方「OAuth Client ID」输入框并保存

快速开始

1. 免费获取 API Key

去 Google Cloud Console 创建项目 → 启用 YouTube Data API v3 → 创建 API 密钥。完全免费，每日 10,000 配额。

2. 添加你的频道

支持频道链接、UCxxx ID、@用户名，自动解析频道信息

3. 一键批量抓取

点「批量抓取所有频道」，自动拉取所有频道的视频数据，聚合为周/月报表

4. 深度指标（可选）

配置 OAuth Client ID 后可获取 CTR、均播时长、展示次数等 Studio 级别数据。同样免费。

所有数据保存在浏览器本地，不上传任何服务器。
